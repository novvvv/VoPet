# VoPet - 단어 해석기 크롬 익스텐션

드래그한 단어를 실시간으로 해석해주는 크롬 익스텐션입니다.

## 기능

- **드래그 해석**: 웹페이지에서 단어를 드래그하면 즉시 해석 팝업이 나타납니다
- **다국어 지원**: 한국어, 영어, 일본어, 중국어 해석 지원
- **API 연동**: Google Translate API를 사용한 정확한 번역

## 설치 방법

1. 이 폴더를 다운로드합니다
2. Chrome 브라우저에서 `chrome://extensions/`로 이동합니다
3. "개발자 모드"를 활성화합니다
4. "압축해제된 확장 프로그램을 로드합니다"를 클릭합니다
5. 이 폴더를 선택합니다

## 사용법

1. 웹페이지에서 해석하고 싶은 단어를 드래그합니다
2. 작은 팝업이 나타나 단어를 해석해드립니다
3. 팝업을 클릭하면 더 자세한 정보를 볼 수 있습니다
4. 익스텐션 아이콘을 클릭하여 설정을 변경할 수 있습니다

## 설정

- **해석 언어**: 원하는 언어로 설정 가능
- **API 키**: Google Translate API 키를 입력하면 더 정확한 번역 제공

## API 키 설정 (선택사항)

더 정확한 번역을 위해 Google Translate API 키를 설정할 수 있습니다:

1. [Google Cloud Console](https://console.cloud.google.com/)에서 프로젝트를 생성합니다
2. Translation API를 활성화합니다
3. API 키를 생성합니다
4. 익스텐션 설정에서 API 키를 입력합니다
